# 🚀 PortSight v1.0.0 — Initial Release

We are excited to announce the first official release of **PortSight**, a high-performance multithreaded port scanner designed for precision and speed.

## ✨ Highlights

### 🔄 Scan Comparison (Differential Analysis)
The standout feature of v1.0.0 is the **Scan Comparison** engine. By comparing your current scan results with a previous baseline, PortSight identifies exactly what has changed in your network infrastructure.
- **OPENED**: New entry points detected.
- **CLOSED**: Previously open ports that are now unreachable.
- **PERSISTENT**: Services remaining stable.

### 🚀 Performance & Scalability
- **Multithreaded Engine**: Powered by `ThreadPoolExecutor` for high-concurrency scanning.
- **Optimized Sockets**: Fine-tuned timeouts and `connect_ex` logic for rapid discovery.

### 🎨 Modern CLI Experience
- **Stylized UX**: Built with the `rich` library for beautiful tables and progress bars.
- **ASCII Branding**: Minimalist professional branding in the terminal.

---

## 🛠️ Features Included
- **Target Resolution**: Domain name to IP mapping.
- **Flexible Port Selection**: Single ports, lists, and ranges (e.g., `80`, `22,443`, `1-1024`).
- **Service Intelligence**: IANA port mapping and banner grabbing.
- **Full Reporting**: Support for JSON and CSV exports.
- **CI/CD Ready**: Automated testing with `pytest` and GitHub Actions.
